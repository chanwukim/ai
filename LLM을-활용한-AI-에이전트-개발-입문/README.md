## [1장 LLM으로 어떤 일을 할 수 있을까?](./chap01/)

생성형 AI(generative AI) : 새로운 데이터를 생성하는 능력에 초점을 맞춘 인공지능.
대규모 언어 모델(Large Language Model, LLM)은 방대한 양의 텍스트를 학습하여 인간의 언어를 이해하고 생성할 수 있다.

LLM의 한계를 보완하는 기술 6가지

-   프롬프트 엔지니어링: 언어 모델의 답변을 최적화하기 위해 입력 프롬프트를 설계하는 것.언어 모델에게 일종의 매뉴얼을 제공하는 지시문
-   파인 튜닝: 이미 학습된 언어 모델에 원하는 분야나 특정 용도에 맞게 추가 데이터를 학습하는 기법
-   RAG(Retrieval Augmented Generation): 필요한 정보를 검색해서 답변할 때 활용하도록 돕는 기술. 쿼리가 들어왔을 때 외부 문서나 데이터베이스에서 관련 정보를 찾고 그 내용을 토대로 답변을 생성하게 만든다.
-   펑션 콜링과 도구 호출: LLM이 외부 API나 직접 만든 함수를 호출하여 그 결과를 바탕으로 답변할 수 있게 하는 기술
-   랭체인: LLM을 활용하여 애플리케이션을 개발하는 프레임워크.
-   랭그래프: 여러 AI 에이전트를 만들어 협업하도록 시스템을 구성하는 방식을 멀티 에이전트라고 한다. 랭그래프를 이용하여 이런 에이전트에게 임무를 부여하고 서로 협업하는 시스템을 개발한다.

## [3장 오픈AI의 APi로 챗봇 만들기](./chap03/)

프롬프트 엔지니어링: GPT 같은 언어 모델이 사용자가 원하는 답변을 하도록 유도하기 위해 문장을 입력하고 효과적으로 설계하는 작업. 사전에 어떤 프롬프트를 입력해 두었는지에 따라 답변이 달라질 수 있다.

원샷 프롬프팅(one-shot prompting): GPT가 원하는 패턴에 맞춰 답변하도록 예시를 한 번 제시해서 유도하는 방식. 여러 번 알려 주는 방식은 퓨샷 프롬프팅(few-shot prompting)

### 멀티턴 대화하기

멀티턴: 여러 번 대화(턴)할 때 이전 대화를 기억하고 적절하게 반응하는 것.

### 스트림릿으로 챗봇 완성하기

https://streamlit.io/

파이썬 프로그램을 웹 기반 UI로 구현할 수 있게 해주는 프레임워크

```bash
pip install streamlit
```

```bash
streamlit run streamlit_basic.py
```

## [5장 회의록을 정리하는 AI 서기](./chap05/)

GPT는 텍스트를 이해하고 처리하는 언어모델이다. 따라서 녹음된 음성 파일을 텍스트로 변환해서 제공해야만 처리할 수 있다.
음성을 텍스트로 변환하는 기술을 STT(Speach-To-Text)라고 한다.
오픈AI는 위스퍼(Whisper) API를 통해 기능을 제공한다.

## [8장 랭체인으로 에이전트 만들기](./chap08/)

랭체인: 언어 모델에 기반한 AI 애플리케이션을 쉽게 개발할 수 있도록 도와주는 프레임워크.
기존에는 언어 모델의 API를 사용해 원하는 기능을 구현하려면 모든 코드를 직접 작성해야 했지만, 랭체인은 이 작업을 간소화할 수 있게 다양한 도구와 모듈을 제공한다.
또한 언어 모델을 쉽게 교체할 수 있다.
